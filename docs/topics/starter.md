# Starter

<warning>
This project is currently in the testing phase and has not yet reached a stable release. Do not use it in production environments.
</warning>

## Introduction

Async Python client for arXiv API with built-in rate limiting and paper downloads.

## Features

- Async paper search & downloads from arXiv
- Built-in rate limiting and concurrency control
- Customizable configuration
- Response parsing
- Type-safe and validation with Pydantic

## Requirements

### System Requirements
- Python 3.9 or higher

### Dependencies
- aiohttp
- pydantic
- loguru
- tenacity
- aiofiles
- platformdirs
- tzdata
- pydantic-settings
